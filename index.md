## 常见问题

### 如何重置或找回密码？

在登录页面点击「忘记密码」链接，输入您注册时使用的邮箱地址，系统将向该邮箱发送一个临时密码。使用临时密码登录后，请前往「我的」页面及时修改密码。

> 如果未收到邮件，请检查垃圾邮件文件夹，或确认邮箱地址是否正确。

---

### 为什么语音录入无法工作？

语音录入需要麦克风权限。请前往 iPhone「设置」→「隐私与安全性」→「麦克风」，确认已为「任务分解」开启麦克风权限。

此外，语音识别需要网络连接，请确保设备已连接到稳定的网络（Wi-Fi 或蜂窝数据）。

---

### 如何添加好友并协作？

1. 进入底部导航的「朋友」页面
2. 点击右上角的添加按钮
3. 输入对方的用户名或邮箱发送好友申请
4. 对方在「朋友」页面确认后即建立好友关系
5. 创建或分解任务时，通过「选人」功能将任务分配给该好友

---

### 任务状态有哪些，如何更新？

| 状态值 | 状态名 | 说明 |
|--------|--------|------|
| 0 | **未完成** | 任务已创建，尚未开始处理 |
| 1 | **工作中** | 正在处理中 |
| 2 | **已完成** | 任务已完成 |
| 3 | **退回** | 任务被退回，需要重新处理 |
| 4 | **终止** | 任务已终止 |

在「需要我处理的」列表中，点击每条任务后面的「处理」按钮，选择对应状态后保存即可更新。

---

### AI 任务整理功能如何使用？

1. 进入语音录入页面，点击录音按钮开始说话（最长支持 2 分钟）
2. 停止录音后，点击「AI 整理」按钮
3. 系统将口述内容发送至 AI 模型进行结构化处理
4. 自动生成任务标题和任务列表
5. 手动编辑后，点击保存将任务提交到系统

> AI 整理依赖网络连接，请确保设备联网。

---

### 数据会保存在哪里？是否安全？

- 任务数据存储在服务端数据库中，通过 **Token 鉴权**保护
- 只有您本人（及您明确授权的好友）才能访问相关数据
- 密码在存储前经过 **SHA-256 算法加密**，服务端不以明文保存任何密码
- 所有接口请求均需通过有效 Token 才能访问，未经授权的请求会被拦截

---

### 如何配置服务器地址（私有部署）？

如您所在企业使用私有化部署版本，请进入 App 的「配置」页面，在服务器地址栏填写您的服务器 IP 或域名（含端口号，如 `http://192.168.1.100:8080`），保存后重新登录即可连接到您的私有服务器。

---

### 实时推送消息无法收到怎么办？

本应用使用 SSE（Server-Sent Events）技术实现实时推送，依赖长连接网络。如果无法收到实时消息，请检查：

1. 网络连接是否稳定（建议使用 Wi-Fi）
2. 尝试退出并重新登录，重建 SSE 连接
3. 检查防火墙或企业网络是否屏蔽了长连接请求
4. 如问题持续，联系服务端管理员确认 SSE 服务是否正常运行

## 功能详细说明

### 我创建的 — 任务列表与创建

页面分为上下两部分：

**上方输入区：**
- 填写任务名称、任务内容、预计完成时间
- 点击执行人右侧图标可从好友列表中选择执行人（不选则默认为自己）
- 点击「保存」提交任务

**下方任务列表：**
- 显示字段：任务名称、创建人、执行人、创建时间、计划完成时间、当前状态
- 点击第一列的层叠图标，可展开查看该任务的所有子任务

---

### 需要我处理的 — 任务分解与状态处理

展示分配给您的所有任务，每条任务右侧有两个操作按钮：

- **分解：** 将当前任务进一步拆分为子任务，填写执行人、任务内容和计划完成时间后保存
- **处理：** 更新任务的当前状态（未完成 / 工作中 / 已完成 / 退回），选择后点击保存即时更新

点击第一列的层叠图标可展开查看该任务下的子任务列表。

---

### 朋友 — 好友管理

好友状态说明：

| 状态 | 说明 |
|------|------|
| **待确认** | 您已发送申请，等待对方确认 |
| **已通过** | 双方已建立好友关系，可相互分配任务 |
| **已拒绝** | 对方拒绝了您的好友申请 |

添加好友时需要知道对方的用户账号。好友关系建立后，在选人组件中即可看到该好友并为其分配任务。

---

### 我的 — 个人信息与账号管理

页面显示您的姓名、邮箱和手机号，并提供两个操作：

- **修改密码：** 输入两次新密码并通过一致性验证，确认后更新
- **修改信息：** 可修改邮箱和手机号，用户名（姓名）注册后不可更改

---

### 语音录入 — 语音 + AI 智能整理

完整使用流程：

```
1. 进入语音录入页面，点击录音按钮开始录制
2. 清晰流畅地描述您的工作内容（最长 2 分钟）
3. 点击停止按钮结束录音
4. 系统自动将语音转为文字显示在文本框中
5. 您可以手动编辑识别文字后，点击「AI 整理」
6. AI 将文字整理为结构化任务列表，包含任务标题和子任务
7. 确认内容后点击保存，批量创建至任务系统
```

## 隐私政策摘要

**我们的承诺：**

| 承诺 | 说明 |
|------|------|
| 🔒 **密码加密存储** | 所有密码经 SHA-256 哈希加密后才存入数据库，服务端不存储明文密码 |
| 🚫 **不售卖个人数据** | 我们不会向任何第三方出售您的个人信息 |
| 📍 **不收集位置信息** | 应用不请求、不收集您的地理位置数据 |
| 🎙️ **录音不留存服务端** | 原始录音文件仅临时存于设备本地，不上传服务端 |

**我们收集的信息：**
注册时提供的姓名、邮箱、手机号；您创建的任务内容；应用使用日志（用于排查技术问题）。

**语音数据说明：**
语音录入功能调用设备本地语音识别能力，录音文件仅临时存储于设备本地。AI 整理时将识别后的**文字**（非原始音频）发送至 AI 服务，原始录音不上传至服务端。

**数据安全：**
所有 API 请求通过 Token 鉴权保护，未授权访问将被拒绝。

**数据共享：**
仅在您明确授权时（如添加好友、分配任务），您的姓名信息才会对特定用户可见。AI 整理功能会将您的任务文字内容发送至 DeepSeek AI 服务进行处理。
